1.Donwload and install Python 3 (make python.exe in the Path Environment Variable)
https://www.python.org/downloads/windows/
Windows XP/Vista: 3.4.4
Windows 7: 3.8.10
Windows 8/8.1/10/11: 3.11.3

2.Download VDF in ReadMe.md (257 files start with http://professional.avira-update.com/update/x_vdf/ ), extract them to the same folder.

3.Modify the folder path (E:\GitHub\AviraCommandLine\) in savapi.conf (2 lines)

4.Install SAVAPI service by savapiinstall.cmd

5.Update SAVAPI/Engine/VDF by savapiupdate.cmd

6.Run scanner by savapiscan.cmd
